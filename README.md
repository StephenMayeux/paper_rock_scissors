# PaperRockScissors

Very simple 1 or 2 player Paper, Rock, Scissors game.

## Installation

    $ gem install paper_rock_scissors

## Usage

In IRB/PRY:

    $ require 'paper_rock_scissors'

And then invoke this module method to start the game.

    $ PaperRockScissors.run
